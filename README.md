# Convert a W3C Recommendation to ISO

This is a first stab at providing an automated way to convert a W3C Recommendation in HTML format to something that's as compliant as possible to an ISO's .docx document.

Tool leverages the [docx](https://docx.js.org/#/) Node.js library.

Relevant links:
- [ISO/IEC Directives, Part 2](https://www.iso.org/sites/directives/current/part2/index.xhtml)
- [Word template for ISO standards](https://www.iso.org/iso-templates.html)
- [W3C's PAS transposition process](https://www.w3.org/guide/process/pas-transposition-process.html)
